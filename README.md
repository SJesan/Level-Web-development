Interfaces এবং Types এর মধ্যে প্রধান পার্থক্য হলো 
--> Interface সাধারণত অবজেক্টের কাঠামো নির্ধারণের জন্য ব্যবহৃত হয়, এবং যখন এক নামের একাধিকবার ঘোষণা করা হয়, সেগুলো স্বয়ংক্রিয়ভাবে একত্রিত হয়ে যায়। 
-->অপরদিকে, Type একত্রিত হয় না। Type আরও নমনীয় এটি primitive type, union type, tuple ইত্যাদি তৈরি করতে পারে,যা Interface এর দ্বারা সম্ভব নয়। 
-->Interface মূলত অবজেক্টের আকার বোঝাতে ব্যবহৃত হয়, যেখানে Type ব্যবহার করা হয় আরও ব্যাপক টাইপ সংজ্ঞার জন্য।

keyof কীওয়ার্ডটি ব্যবহৃত হয় একটি অবজেক্টের কীগুলিকে একটি union টাইপ হিসেবে পাওয়ার জন্য। উদাহরণস্বরূপ, যদি User টাইপের মধ্যে name, age, এবং email থাকে, তাহলে keyof User লিখলে name | age | email টাইপ তৈরি হয়। এটি কোডকে আরও টাইপ-সেফ করে তোলে, কারণ এর ফলে পরবর্তীতে কম্পাইলার এই কীগুলোকে সঠিকভাবে শনাক্ত করতে সক্ষম হয়।
